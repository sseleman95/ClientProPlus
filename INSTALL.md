# To run the

1. To clone the repository in the project portfolio:

    ```bash
    git clone https://github.com/sseleman95/ClientProPlus.git
    cd ClientProPlus
    ```

2. To run the:

    ```bash
    python -m http.server 8000 # Python
    npx serve . # Node.js
    ```

3. Then open it in your browser:

    ```text
    http://localhost:8000
    ```
