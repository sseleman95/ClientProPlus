Introducción a ClientPro+

ClientPro+ es un sistema completo de gestión de clientes y financiamiento diseñado específicamente para negocios que ofrecen planes de pago a sus clientes.

¿Por qué una PWA?

Progressive Web App (PWA) es la tecnología moderna que combina lo mejor de las aplicaciones web y móviles. ClientPro+ como PWA ofrece:

  • Funciona sin conexión a internet
  • Se instala en cualquier dispositivo (PC, móvil, tablet)
  • No requiere tiendas de aplicaciones
  • Actualizaciones automáticas
  • Mayor seguridad y rendimiento

    • Uso en PC

    Instálelo en su computadora como una aplicación nativa, accesible desde el escritorio o menú de inicio.

    Funciona Offline

    Continue trabajando incluso sin conexión a internet. Los datos se sincronizan automáticamente cuando se reconecta.

    Sincronización

    Todos sus datos se sincronizan entre dispositivos si utiliza nuestro servicio en la nube (opcional).

    ¿Qué es una PWA y cómo funciona?

    Una Progressive Web App (PWA) es una aplicación web que utiliza tecnologías modernas para ofrecer una experiencia similar a una aplicación nativa.

    Característica Aplicación Tradicional PWA (ClientPro+)
    Instalación Requiere tienda de aplicaciones Directamente desde el navegador
    Actualizaciones Manual desde la tienda Automáticas en segundo plano
    Espacio Ocupa mucho espacio Mínimo espacio requerido
    Offline Depende de la app Funcionalidad completa offline
    Multiplataforma Versiones separadas Una versión para todos los dispositivos

    Experiencia personalizada

    Contenido y recomendaciones adaptados a tus intereses y comportamiento.

    ¿Cómo se instala una PWA?

    La instalación es muy sencilla. Cuando accede a ClientPro+ desde su navegador (Chrome, Edge, Safari), verá un ícono de instalación. Con un solo clic, la aplicación se instalará en su dispositivo.

    En PC aparecerá en su menú de inicio o escritorio. En móviles, aparecerá en la pantalla principal como cualquier otra aplicación.

    3

    Instalación Local en su PC

    Siga estos sencillos pasos para instalar y ejecutar ClientPro+ en su computadora local:

    1

    Descargue los Archivos

    Recibirá un archivo ZIP con todos los archivos necesarios. Descomprímalo en la carpeta de su preferencia.

    ClientPro+-v1.0.zip Contenido: - index.html (archivo principal) - /css (estilos) - /js (funcionalidades) - /img (imágenes) - manifest.json (configuración PWA) - service-worker.js (para modo offline)
    2

    Ejecute con un Servidor Local

    Para que la PWA funcione correctamente, necesita ejecutarse desde un servidor local:

    Opción A: Servidor Simple (Recomendado)

    # Si tiene Python instalado: python -m http.server 8000
    # Si tiene Node.js instalado: npx serve .
    # Luego abra en su navegador: http://localhost:8000

    Opción B: XAMPP/WAMP (Para avanzados)

    Copie la carpeta descomprimida dentro de la carpeta htdocs de XAMPP y acceda mediante http://localhost/clientpro

    3

    Instale como Aplicación

    Una vez que la aplicación se cargue en su navegador:

  • En Chrome/Edge: Busque el ícono "Instalar" en la barra de direcciones
  • En Firefox: Vaya al menú y seleccione "Instalar sitio"
  • En Safari: Use el botón compartir y "Añadir a pantalla de inicio"

    • Consejo importante

    Para el modo offline completo, asegúrese de que el Service Worker esté correctamente registrado. Esto sucede automáticamente la primera vez que accede a la aplicación.

    Uso del Sistema

    ClientPro+ está diseñado para ser intuitivo y fácil de usar. Aquí le mostramos el flujo de trabajo principal:

    Pantalla de Login
    Dashboard Principal
    Registro de Cliente
    Nuevo Financiamiento

    Flujo de Trabajo Típico:

    1. Registrar Cliente

    Complete los datos detallados del cliente, referencias y documentación. El sistema genera un ID único automáticamente, con notas internas y observaciones.

    2. Crear Financiamiento

    Seleccione productos o crea registro detallado del producto, configure el plan de pagos y genere el contrato automático, dandole acceso a cálculo automático de subtotales, impuestos y totales

    3. Seguimiento de Pagos

    El sistema crea automáticamente el calendario de pagos y envía recordatorios.

    4. Monitoreo y Reportes

    Visualice el estado de todos los financiamientos, clientes atrasados y métricas del negocio.
